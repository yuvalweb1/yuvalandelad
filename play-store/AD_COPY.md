# Ad Copy — paste-ready

Three **English** visuals are generated by `scripts/generate-ads.mjs`:

- `play-store/ad-square-1080-en.png` — Instagram / Facebook feed
- `play-store/ad-story-1080x1920-en.png` — Story / Reel / TikTok cover
- `play-store/feature-graphic-1024x500-en.png` — Play Store feature graphic

**Hebrew visuals must be built in Canva** — librsvg (the SVG renderer
sharp uses) doesn't correctly bidi-shape Hebrew text or place mid-line
accent spans in RTL flow. The Hebrew copy below is paste-ready for that.

Recipe for the Canva Hebrew creatives: open the corresponding English PNG
as a Canva "starting point", swap the text frames with the Hebrew copy
below, ensure each Hebrew text frame is set to RTL direction. Same
gradient, same dark Verdict card layout. ~10 min per size.

Pair them with the copy below.
